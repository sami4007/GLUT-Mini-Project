# 🏛️ CUET Gate Design using OpenGL (GLUT)

A Computer Graphics course project developed in **C++ using OpenGL and GLUT** that recreates the **main entrance gate of Chittagong University of Engineering & Technology (CUET)** with animated environmental elements.

The project demonstrates the use of **2D computer graphics**, **OpenGL primitives**, **transformations**, **animation**, and **interactive rendering** to create a realistic campus entrance scene.

---

## 📖 Project Overview

This project recreates the iconic **CUET Main Gate** in a 2D OpenGL environment. The scene includes detailed architectural structures along with animated objects such as moving vehicles, drifting clouds, a moving sun, waving grass, and a security guard performing a salute animation.

The primary objective of this project is to apply fundamental computer graphics concepts learned during the Computer Graphics course.

## ▶️ How to Run

### Requirements

* C++ Compiler (GCC/MinGW or MSVC)
* OpenGL
* GLUT / FreeGLUT
* Code::Blocks (Recommended)

### Steps

1. Clone the repository

```bash
git clone https://github.com/sami4007/GLUT-Mini-Project.git
```

2. Open the project using Code::Blocks.

3. Ensure OpenGL and GLUT libraries are properly configured.

4. Build the project.

5. Run the application.
